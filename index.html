About AuraRead

This is a Book recommendation System based on your current Mood. Traditional book recommendation systems suggest titles based on genres or popularity, but they often miss out on one critical factorâ€”how you feel. AuraRead not only finds books that spark your interest, but also books that resonate with your current mood. We used web scrapers to extract mood from book descriptions and reviews, integrating these moods into our dataset. Our model uses collaborative filtering, enhanced with matrix factorization techniques, to predict which books might interest you based on your previous interactions and current mood. Happy reading!

Deployment Steps

  1. Download or Clone the repository: Clone the project using the command git clone https://github.com/prathyushapolepalli-tamu/AuraRead.git
  2. Environment Setup: From the project folder, create and activate the Python environment.
    • Create environment: python3 -m venv venv (if virtual environment is not already set)
    • Activate environment: source venv/bin/activate
  3. Library Installations: Install necessary libraries using the command pip install flask torch pandas scikit-learn
  4. Execution: Bring up the application by running ./venv/bin/python app/app.py
  5. Access the Application: Access the application from http://127.0.0.1:5000 and try out our product.
